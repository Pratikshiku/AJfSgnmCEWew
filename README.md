### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、餐厅、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 餐厅信息
5. 美食信息
6. 美食文化
7. 公告资讯
8. 后台管理
9. 购物车
10. 个人中心
11. 我的订单
12. 我的地址
13. 我的收藏


**餐厅功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 餐厅信息管理
6. 美食信息管理
7. 订单管理



**管理员功能：**
1. 登录
2. 个人中心(修改密码、个人信息)
3. 餐厅管理
4. 用户管理
5. 餐厅信息管理
6. 菜系分类管理
7. 美食信息管理
8. 美食文化管理
9. 系统管理（关于我们、系统简介、轮播图管理、公告资讯）
10. 订单管理
